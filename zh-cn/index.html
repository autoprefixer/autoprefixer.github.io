Autoprefixer 是什么

Autoprefixer 在线 - 原生 Autoprefixer 在线 REPL。 它解析您的 CSS 并使用 Can I Use 中的值 将浏览器供应商前缀添加到 CSS 规则中。它由谷歌推荐，并在 Twitter 和淘宝使用。

它是如何工作的

Autoprefixer 使用有关浏览器受欢迎程度的数据及浏览器对供应商前缀支持的数据。 根据这些信息来增加或删除前缀。 它也可以帮助您获得 animations, transition, transform, grid, flex, flexbox等规则的前缀。

浏览器过滤

您可以配置在转换中为哪些浏览器生成前缀。有关语法的更多详细信息详见文档。常见问题解答在文中已列出。

选项

IE 浏览器的 Grid 布局 polyfill 已启用，在 autoprefixer 中的默认不是这个。
