Qu'est-ce que c'est

Autoprefixer en ligne — web repl pour Autoprefixer. Parse votre CSS et ajoute les préfixes vendeurs aux règles CSS à partir de Can I Use. Autoprefixer est recommandé par Google et utilisé par Twitter et Taobao.

Comment ça marche

Autoprefixer utilise des données sur la popularité des navigateurs et leur support pour les préfixes vendeurs. À partir de ces informations, il arrange et supprime les préfixes. Il peut vous aider à avoir des préfixes pour: les animations, les transitions, transform, grid, flex, flexbox et autres.

Filtrage des navigateurs

Vous pouvez configurer pour quels navigateurs il faut générer les préfixes dans la distribution. Plus de détails sur la syntaxe dans la documentation. Il y en a aussi dans la FAQ.

Options

Le polyfill pour la mise en page des grid IE est activé, ce qui n'est pas le cas par défaut dans Autoprefixer.
