O que é isso

Autoprefixer online — web repl para o Autoprefixer original. Ele analisa seu CSS e adiciona o vendor prefixos para regras CSS usando valores de Eu posso usar. Isso é recomendado pelo Google e usado pelo Twitter e Taobao.

Como funciona

O Autoprefixer usa dados sobre a popularidade dos navegadores e suporte para prefixos de vendor por navegadores. Com base nessas informações, ele organiza e exclui os prefixos. Ele pode ajudá-lo a obter prefixos para: animações, transição, transformação, grade, flex, flexbox e outros.

Filtro do navegador

Você pode configurar para quais navegadores gerar prefixos na distribuição. Mais detalhes sobre a sintaxe estão na documentação. Há também um bloco no FAQ.

Opções

O polyfill de layout de grade do IE está habilitado, o que não é por padrão no autoprefixer.
